Per la realizzazione della chatroom sono stati usati:

-un server: file.dart preso dalla repository comune gitlab al quale sono state apportate modifiche per un funzionamento migliore;

-un client: progetto flutter che ha al suo interno più classi che a loro volta implementano una connessione con google (appoggiata da firebase),
più schermate (di login, start chat, chatroom).

Il protocollo usato per la comunicazione server-client è di tipo TCP.
